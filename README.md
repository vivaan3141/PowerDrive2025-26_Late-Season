## Team 13777X – Push Back (2025–2026) Season

This repository contains the complete programming for **VEX Robotics Team 13777X** during the **2025–2026 Push Back** season. 
The code is designed to increase efficiency, consistency, and scoring in both autonomous and driver-controlled periods.

## Competition Code
- **Autonomous:** Pre-programmed paths during autonomous period.
- **Driver Control:** Controls for driver during VEX Match
- **Sensors:** Advanced use of inertial and distance sensors.
- **Late-Season Changes:** Code updates to optimize and organize program.

## Code Structure
- Written in C++ Language, which is great for embedded systems (hardware integration).
- `/src` – Core robot logic + control systems  
- `/json` – Integration with VEX library
- `/include` – Constants, configurations, and coordination between src files
